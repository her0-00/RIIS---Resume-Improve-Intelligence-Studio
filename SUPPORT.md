# 💬 Support Center & Community Guidelines

Thank you for using **IRIS (Improve Resume Intelligence Studio)**! We are committed to building a vibrant, supportive, and active developer community. Whether you have questions, found a bug, want to suggest features, or just want to chat with fellow builders, we've got you covered.

> [!NOTE]
> Please check if your question has already been answered in our [README.md](README.md), [CONTRIBUTING.md](CONTRIBUTING.md), or the existing [GitHub Issues](https://github.com/her0-00/IRIS---Improve-Resume-Intelligence-Studio/issues) before opening a new thread.

---

## 🚀 How to Get Help

We provide different channels tailored to specific needs. Please choose the appropriate channel to ensure you get the fastest and most relevant assistance:

| Channel | Purpose | Response Expectation | Link |
| :--- | :--- | :--- | :--- |
| **💡 GitHub Discussions** | General questions, configuration help, integration advice, and general showcase of your CV templates. | **1–3 Days** (Community-backed) | [Open Discussions](https://github.com/her0-00/IRIS---Improve-Resume-Intelligence-Studio/discussions) |
| **🐛 GitHub Issues** | Reporting reproducible bugs, performance issues, or app crashes in either the React frontend or the Python ReportLab backend. | **24–48 Hours** (Maintainer response) | [Report a Bug](https://github.com/her0-00/IRIS---Improve-Resume-Intelligence-Studio/issues/new?template=bug_report.md) |
| **✨ Feature Requests** | Proposing new themes, new PDF layout options, ATS analyzer rules, or structural features. | **Reviewed Weekly** | [Request Feature](https://github.com/her0-00/IRIS---Improve-Resume-Intelligence-Studio/issues/new?template=feature_request.md) |
| **🛡️ Security Reports** | Reporting security issues or vulnerabilities in our dependencies or processing pipeline. | **Immediate / Within 12 Hours** | See [SECURITY.md](SECURITY.md) |
